Padding VS Margin VS Border

This blog should have those very answers that you are looking for!

Margin is the outer space around the box, border is a visible or invisible line surrounding the box and padding is the space between the content and the border.

The picture below is an example of this.

People confuse margin with padding because they both serve very similar purposes and they both neighbor with the border. However, they both have different principles.
Margin is an external element, which means it exists outside of the box. It's job is to control the space between different elements.
Padding is inside the box, so it controls the inside spaceing, which can\ control how much the background color is visible around the content.

In this picture, the padding is set to 100px on all sides, increasing the spacing inside the box structure.

In this picture, the opposite is true. The margin is set to 100px and the padding is set to 0. This shows the effect is identical, which is why people get confused between margin and padding.

When would we use margin?

There are three popular cases where CSS margin can be useful to the developer.

  • Change the element's position on the page.
    To set the element to the top, bottom, left or right of the page, or even the centered position. To reach the centered position, make sure the width of the page is set and set the margin value to "auto."
  • Add distance between two elements in the page.
    CSS margins are used to make the element's arrangement visually appealing by adding spacing between them. This way, two images or an image and text won't be joined to each other, therefore distracting the user.
  • Overlapping elements.
    The negative margin value helps to overlap an element over the top of another element space. In the image below, you can see how one image overlaps the other with the help of a margin command. This creates a broken grid layout.

When would we use padding?

CSS padding can be useful to achieve certain effects. Here are some of the most common ones.

  • Increase space between content and it's border.
    Padding gives an opportunity to put breathing space around the element. For example, additional space around a button can improve the readablilty, as well as its general design.
  • Adjust the size of the element.
    Although padding doesn't influence the size of the content it can increase or decrease the space around the element, making it more clickable or visable. This is especially popular when creating product cards and buttons.

How to edit margins and paddings?

There are two popular ways to edit margin or padding of the element: through code or by using a page builder. I will only be talking about how to be doing it through code, as that is what I am learning here at Dev Academy.

To edit CSS margin or padding through code, open your stylesheet and find the class of the element that you want to change the margin or padding for, edit your values and save. While this does sound easy, it can take some time to find the right class and employ your desired value.
